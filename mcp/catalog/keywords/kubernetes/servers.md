# kubernetes MCP Servers

Select only servers that directly match the current request. If a server matches, read its `mcp/servers/<server>/SKILL.md` and use it only within MCP safety rules.

- **Azure Kubernetes Service (AKS)** (`aks`) — `docker`, `reviewed`: Azure Kubernetes Service (AKS) official MCP server
- **mcp** (`com.googleapis.container-mcp`) — `hosted_https`, `trusted`: Provides read access to your GKE and Kubernetes resources.
