# Knowledge Base Maintenance Workflow

Workflow id: `knowledge-base-maintenance`

Use this workflow to audit, update, deprecate, and govern internal knowledge articles, decision logs, runbooks, and onboarding docs.

- Manifest: `workflows/documentation/knowledge-base-maintenance/workflow.yaml`

## Coordinator Notes

Follow the manifest for stage order, role delegation, gates, and acceptance criteria.
