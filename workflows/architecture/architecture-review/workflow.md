# Architecture Review Workflow

Workflow id: `architecture-review`

Use this workflow to evaluate an architecture decision, design, or proposal before implementation or major commitment.

## Files

- Manifest: `workflows/architecture/architecture-review/workflow.yaml`
- Common schemas: `workflows/templates/schemas/`
- Common artifact templates: `workflows/templates/artifacts/`

## Coordinator Notes

This workflow reviews an existing or proposed design. Use `adr-decision` when the primary output is choosing between options and recording a decision.

Follow the manifest for stage order, role delegation, gates, and acceptance criteria.
